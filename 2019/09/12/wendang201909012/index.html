辅助运输二期需求文档

发表于 2019-09-12 | 分类于 文档

基础功能(预计完成时间10月12日)

订单系统

服务端

  • 任务结束后修改订单状态（同自动结束任务）
  • 查询终端合法状态
  • 查询终端合法状态
  • 车队派单后推送到平板
  • 获取司机信息
  • 绑定司机与车辆
  • 行车导航数据
  • 收车解除司机与车辆绑定

    客户端

  • 查询终端合法状态
  • 获取司机信息
  • 绑定司机与车辆
    • 如果之前有绑定的司机 自动替换
  • 弹出实时订单
  • 出车&拒绝
  • 绘制行车导航路径
  • 结束任务
  • 已完成任务列表
  • 已拒绝任务列表
  • 未完成任务列表
  • 收车解除司机与车辆绑定
    • 回到初始界面

    视频通话

    服务端

  • 调度到平板呼叫视频通话
  • 行车记录监控功能

    客户端

  • 调度到平板呼叫视频通话
  • 行车记录监控功能
  • 通过话筒通话
  • 通过底座的外放

车载底座按键功能

客户端

  • 出车 长按F1键10秒
  • 收车 长按F1键10秒
  • 进入菜单 双击F3键
  • 方向键
  • 确定取消键

扩展功能

订单系统

  • 司机与车辆的匹配

视频通话

  • 语音对讲通话（关闭视频）
  • 平板到调度呼叫视频通话
  • 语音广播功能

其他功能

  • 领导终端APP
  • 司机终端APP
  • 数据分析统计报表

边缘功能

周边障碍物预警模块

客户端

  • 判断行驶方向
  • 判断障碍物前后位置
  • 障碍物距离去抖动：抖动范围：$Range \leq 30cm$

    服务端

  • 筛选碰撞消息符合条件的消息
    • 筛选障碍物类型
      • 固定障碍物：红绿灯、路口等
      • 移动障碍物：车辆、人员等
    • 筛选位移范围： $Range_i \leq Distance \leq Range_j$
    • 筛选方向（详见寻路算法）
    • 筛选时间范围 $Time \nleq \Delta t$
  • 调用寻路算法找到当前车辆行驶的路径
    • 获取附近障碍物轨迹坐标集合
    • 计算附近障碍物距离锚点（井口）位移
    • 确认障碍物在与当前车辆的位置关系
